# TODO / Tech debt

Известные ограничения и идеи на будущее.

## Tech debt

- **Бесконечный откат действий.** Сейчас «Отменить» откатывает последнее неотменённое
  действие по аккаунту, и так можно пройти всю историю назад (журнал `actions` не ограничен
  по времени/количеству). Нужно согласовать с владельцем желаемое поведение, например:
  - откатывать только последнее действие (без цепочки), или
  - разрешать откат только в течение N минут после действия, или
  - разрешать откат только в рамках текущей смены/дня.
  Ограничение задаётся в `backend/app/service.py` (`undo_last`) и в UI (`useGuests.js`).

## Идеи

- MAX-версия мини-аппа (сейчас только Telegram; код фронта абстрагирован под мультиплатформу).
- Тесты (pytest для доменной логики начисления/отката).
- Экспорт статистики (CSV) для владельца.
